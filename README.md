# My-Living-World-RPG

Um RPG por turnos desenvolvido em Python.

## Sobre

My Living World RPG é um projeto pessoal criado com o objetivo de aprender desenvolvimento de jogos e programação. O diferencial do projeto é um mundo vivo, onde monstros evoluem, batalham entre si e continuam existindo independentemente do jogador.

## Funcionalidades atuais

- Sistema de combate por turnos
- Sistema de entidades
- Sistema de XP e níveis
- Sistema de armas
- Organização modular do projeto

## Planejamento

- Sistema de save
- Interface com Tkinter
- Orientação a Objetos
- Classes
- Sistema de buffs
- Modo infinito
- Modo mapa
- Mundo vivo
- Necromante
- Migração futura para C# e Unity

## Tecnologias

- Python 3
